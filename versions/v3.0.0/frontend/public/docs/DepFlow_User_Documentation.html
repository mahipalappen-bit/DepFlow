Welcome to DepFlow, your comprehensive dependency management platform designed for modern development teams. This guide will help you get started with managing your project dependencies efficiently.

What is DepFlow?

DepFlow is an enterprise-grade dependency management platform that helps development teams:

  • Track and manage project dependencies
  • Monitor dependency status and priority
  • Collaborate effectively across teams
  • Receive automated notifications
  • Export and analyze dependency data
üí° Quick Tip: DepFlow works best when all team members are actively using the platform to update dependency statuses and collaborate on project planning.

üîß Managing Dependencies

Adding New Dependencies

Create a New Dependency

  1. Click the "Add Dependency" button
  2. Fill in the required information:
    • Name: Dependency identifier (required)
    • Description: Detailed explanation (minimum 10 characters)
    • Team: Select responsible team
    • Status: Choose from Not Started, In Progress, Blocked, Completed
    • Priority: Set as High, Medium, or Low
  3. Click "Add Dependency" to save

üîê Permissions & Access Control

User Roles

DepFlow implements Role-Based Access Control (RBAC) with two main roles:

üëë Admin Users

  • View all dependencies
  • Edit any dependency
  • Delete any dependency
  • Access all system features

üë§ Regular Users

  • View all dependencies
  • Create new dependencies
  • Edit own dependencies only
  • Delete own dependencies only

Ownership Rules

  • Dependencies are owned by their creator
  • Regular users can only modify dependencies they created
  • Admin users have full access to all dependencies
  • All users can view and filter all dependencies
üí° Note: The interface remains consistent for all users - permissions are enforced at the action level.

üìß Email Notifications

üîß Troubleshooting

Common Issues

Login Problems

  • Ensure your email and password are correct
  • Check if Caps Lock is enabled
  • Try refreshing the page and logging in again

Can't Edit Dependencies

  • Regular users can only edit dependencies they created
  • Check if you're the original creator of the dependency
  • Contact an administrator if you need to edit someone else's dependency

Missing Dependencies

  • Check your applied filters and search terms
  • Click "Clear Filters" to see all dependencies
  • Ensure you're not filtering by a specific team or status

Email Notifications Not Received

  • Check your spam/junk email folder
  • Verify your email address in your profile
  • Contact your administrator to check email configuration

Browser Compatibility

  • Safari 12+
  • Edge 79+
